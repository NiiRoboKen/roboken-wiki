# 2. RC回路
- R (抵抗)
	- LEDを点灯する回路の制作
- C (コンデンサ)
	- セラコンと電解の違いの説明
	- セラミックコンデンサ
		- 磁石を用いて人為的にノイズを発生させて、パスコンでノイズが取り除かれてるかをオシロで観測する実験
	- 電解コンデンサ
		- 逆接による爆発は見せる
